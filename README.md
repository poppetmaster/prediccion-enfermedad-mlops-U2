# Predicción de Estado de Enfermedad - MLOps Unidad 2

## Descripción del problema

Este repositorio contiene una solución simulada para apoyar al personal médico en la predicción preliminar del estado de enfermedad de un paciente a partir de datos básicos de síntomas, edad, presencia de fiebre y antecedentes de enfermedad base.

## Propósito del repositorio

El propósito del repositorio es aplicar buenas prácticas de MLOps mediante el uso de GitHub, ramas, Pull Requests, pruebas unitarias, Docker y GitHub Actions.
